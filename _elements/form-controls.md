<p class="usa-font-lead">See <a href="https://playbook.cio.gov/designstandards/form-controls/">US Web Design Standards</a> for design description.</p>

The SAM UIKit allows front-end designers and developers to quickly generate the form controls described by the USWDS. There are three groupings of form controls:

1. text input,
2. select,
3. date input.

Text input allows you to generate a text, email, or email <code>input</code> type, as wells as a <code>textarea</code>. The select form control allows you to generate <code>select</code>, <code>checkbox</code>, and <code>radio</code> form controls. The date input form control allows you to genrate the USWDS date input form control.
